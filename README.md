# Focus Timer

A simple, installable countdown timer for focused work, study, cooking and reminders. Works fully offline once loaded.

**Live app:** https://ultra-madness.github.io/focus-timer/

## Features
- Quick presets (Pomodoro 25, break 5, long break 15, 1/3/10/45/60 min) plus custom H:M:S
- Start / pause / resume / reset, and +1 minute on the fly
- Alarm sound, browser notification, vibration, and screen flash when time is up
- Tab-title countdown, dark / light theme, keep-awake, and auto-repeat
- Keyboard shortcuts: Space (start/pause), R (reset), + (add minute), F (fullscreen)
- Installable PWA (Add to Home Screen / Install app) and works offline

Built as a single self-contained `index.html` with no external dependencies.
